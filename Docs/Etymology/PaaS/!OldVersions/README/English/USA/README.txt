# UnSaaSS Documentation

## Etymology

### PaaS

PaaS commonly stands for `Platform as a Service` it is just another name for [`SaaSS`](/Docs/Etymology/SaaSS/) which has many names it goes by. In this case, it should either go by [`SaaSS`](/Docs/Etymology/SaaSS/) or [`PaaSS`](/Docs/Etymology/PaaSS/)
